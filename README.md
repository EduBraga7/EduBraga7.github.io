# Cartao de Visitas Online

Repositorio do meu site pessoal publicado no GitHub Pages.

Site: https://EduBraga7.github.io

## O que este projeto entrega
- Portfolio pessoal em layout de secoes (nao mais card unico)
- Conteudo profissional orientado ao curriculo
- Download direto do curriculo
- Download de curriculo em ingles na versao EN
- Versao em portugues e ingles
- Link direto para projeto Ticker Tracker
- Links para Moto Tracker e Diario de Filmes e Series
- Secao de resultados e impacto profissional
- Links para perfis profissionais
- Layout responsivo para mobile e desktop
- Melhorias de acessibilidade (foco visivel e respeito a reduced motion)

## Tecnologias
- HTML5
- CSS3
- Font Awesome (icones)
- Google Fonts

## Estrutura
- index.html: versao em portugues
- en.html: versao em ingles
- styles.css: design, responsividade e animacoes
- foto-perfil.jpg: imagem de perfil
- curriculo-eduardo-braga.docx: arquivo para download
- curriculo-eduardo-braga-pt.html: curriculo em portugues com layout profissional e pronto para PDF
- resume-eduardo-braga-en.html: curriculo em ingles com layout profissional e pronto para PDF

## Como personalizar rapido
1. Edite nome, bio e links em index.html.
2. Mantenha en.html alinhado com os mesmos dados em ingles.
3. Troque a foto em foto-perfil.jpg.
4. Atualize curriculo-eduardo-braga.docx quando houver novas experiencias.
5. Ajuste paleta e tipografia nas variaveis CSS em styles.css.
