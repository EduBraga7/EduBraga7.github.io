Bem-vindo ao repositório do meu site e portefólio pessoal. Este é o espaço onde documento a minha evolução na área de Análise e Desenvolvimento de Sistemas e centralizo os projetos em que aplico os meus conhecimentos de programação.

## 🎯 O Propósito
A ideia deste site é ir muito além de um currículo estático. Ele serve como o meu laboratório pessoal para testar interfaces, focar no **minimalismo** e na usabilidade, e partilhar as soluções que construo — desde aplicações web para análise de dados até páginas otimizadas para conversão de negócios reais.

## 🛠️ Tecnologias e Ferramentas

Este projeto foi construído utilizando tecnologias *core* da web, sem o excesso de *frameworks*, para garantir total controlo, performance e um código limpo:
