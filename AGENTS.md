# Instrucoes de atualizacao do portfolio

Este arquivo define como o assistente deve aplicar qualquer atualizacao solicitada pelo usuario neste projeto.

## Regra principal
Sempre que o usuario pedir qualquer alteracao de conteudo, aplicar a mudanca em todos os pontos equivalentes:

1. Portfolio em portugues: index.html
2. Portfolio em ingles: en.html
3. Curriculo em portugues: curriculo-eduardo-braga-pt.html
4. Curriculo em ingles: resume-eduardo-braga-en.html

## Regra de consistencia
- Nao deixar textos divergentes entre PT e EN quando o conteudo for equivalente.
- Se alterar habilidades, experiencias, projetos, contato, objetivo ou apresentacao, sincronizar em PT e EN.
- Manter tom profissional, com acentuacao correta no portugues.

## Curriculos em PDF
Quando houver alteracao de conteudo em qualquer curriculo HTML:

1. Regenerar curriculo-eduardo-braga.pdf a partir de curriculo-eduardo-braga-pt.html.
2. Regenerar resume-eduardo-braga-en.pdf a partir de resume-eduardo-braga-en.html.
3. Conferir se os botoes de download continuam apontando para os PDFs corretos.

## Checklist rapido por atualizacao
- Atualizar index.html
- Atualizar en.html
- Atualizar curriculo-eduardo-braga-pt.html
- Atualizar resume-eduardo-braga-en.html
- Regenerar curriculo-eduardo-braga.pdf
- Regenerar resume-eduardo-braga-en.pdf
- Revisar acentuacao do PT
- Validar links e botoes

## Regra de execucao
- Aplicar alteracoes direto, sem pedir confirmacao extra, salvo quando faltar informacao essencial.
- Preservar o estilo visual atual, alterando apenas o necessario para atender ao pedido.
